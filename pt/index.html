5 ★ Dados Abertos

Tim Berners-Lee, o inventor da Web e fundados do conceito de Dados Linkados, sugeriu um esquema de 5 estrelas/a> para Dados Abertos. Aqui estão exemplos para cada uma das estrelas e explicações sobre custos e benefícios que acompanham cada estágio.

Custos & Benefícios ...

Quais são os custos & benefícios de ★ Dados na Web?

Como consumidor ...

  • ✔ Você pode ver os arquivos.
  • ✔ Você pode imprimir os arquivos.
  • ✔ Você pode guardá-los localmente..
  • ✔ Você pode inserir os dados em qualquer outro sistema
  • ✔ Você pode modificar os dados do jeito que você achar melhor.
  • ✔ Você pode compartilhar os dados com quem você quiser.

Como publicador ...

  • ✔ É simples publicar.
  • ✔ Você não precisa repetir infinitamente para as pessoas que elas podem usar os seus dados
“

É muito bom ter dados acessíveis na Web sobre uma licença aberta, (como a PDDL, ODC-by or CC0), entretanto, os dados estão presos em um documento. É difícil extrair dados de documentos.

”

Quais são os custos & benefícios de ★★ Dados na Web?

Como consumidor, você pode fazer o que você quiser com ★ Dados na Web e ainda por cima:

  • ✔ Você pode processar os dados diretamente com software proprietário para agregar, explorar, visualizar, etc
  • ✔ Você pode exportar para outro formato estruturado

Como publicador ...

  • ✔ É simples publicar
“

Esplêndido! Os dados estão acessíveis na Web e estruturados! (ou seja, legíveis por máquinas), entretanto, os dados ainda estão enjaulados em um documento. Para retirar os dados deste documento, você depende de software proprietário.

”

Quais são os custos & benefícios de ★★★ dados na Web?

Como consumidor,você pode fazer o que quiser com ★★ dados na Web e ainda:

  • ✔ Manipular os dados da maneira que você quiser, sem precisar comprar software proprietário para isso.

Como publicador...

  • ⚠ Você pode precisar de conversores para exportar os dados de formato proprietário para formato livre.
  • ✔ Ainda é mais simples de publicar.
“

Excelente! Os dados estão disponíveis na Web e agora todos podem usar os dados facilmente. No entanto, ainda são dados na web, mas não estão contextualizados. Precisamos chegar em data in the Web.

”

Quais são os custos & benefícios de ★★★★ Web data?

Como consumidor, você pode fazer o que quiser com ★★★ Dados na Web e também pode:

  • ✔ Linkar de qualquer outro lugar na Web.
  • ✔ Pode colocar nos seus bookmarks
  • ✔ Utilizar parte dos dados para reuso
  • ✔ Você pode reutilizar ferramentas e tecnologias, ainda que eles só entendam partes dos padrões que o publicador utilizou.
  • ⚠ Entender a estrutura de um RDF pode ser pode ser mais difícil do que trabalhar com dados tabulares (Excel/CSV) ou em árvore (XML/JSON)
  • ✔ Você pode combinar dados em segurança, com outros dados. URIs são um esquema global e padronizado.

Como publicador ...

  • ✔ Você tem controle granular sobre os dados e pode otimizar o seu acesso.
  • ✔ Outros publicadores de dados podem linkar para os seus dados, promovendo as 5 estrelas!
  • ⚠ Você vai gastar algum tempo regularizando seus dados.
  • ⚠ Você vai precisar especificar URIs para os dados e pensar em como representar os dados.
  • ⚠ Você vai precisar criar ou reaproveitar padrões existentes
“

Maravilha! Agora são dados NA Web. Os itens mais importantes tem URI's e podem ser compartilhados na Web. Um jeito nativo de representar dados é usar o RDF, mas existem outros formatos, como o ATOM, que podem ser convertidos e mapeados, se necessário.

”

Quais são os custos & benefícios de ★★★★★ Web data?

Como consumidor, você pode fazer o que você quiser com ★★★★Dados na Web e ainda:

  • ✔ Pode descobrir mais dados relacionados enquanto consome de uma fonte de dados.
  • ✔ Pode aprender diretamente sobre o schema dos dados.
  • ⚠ Agora você tem que lidar com links quebrados, ou erros 404 em páginas web.
  • ⚠ Apresentar dados em links arbitrários é tão arriscado quanto deixar pessoas incluirem conteúdo de outros websites em seu site. Cuidado e bom senso ainda são necessaários.

Como publicador de dados ...

  • ✔ Você torna seus dados encontráveis na Web.
  • ✔ Você aumenta o valor dos seus dados.
  • ✔ Sua própria organização pode tirar vantagem consumindo dos dados linkados para adquirir inteligência.
  • ⚠ Você vai precisar investir recursos para linkar seus dados a outros dados na Web.
  • ⚠ Você pode precisar reparar links quebrados.
“

Brilhante! Agora estamos falando de Dados na Web linkados a outros dados. Ambos, o consumidor e o publicador de dados se beneficiam dos efeitos de rede..

”
